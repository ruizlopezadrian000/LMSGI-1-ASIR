1. Objetivo del Proyecto.

Elaboración de un proyecto web, empleando las distintas etiquetas de HTML5 y aplicando atributo style con propiedades CSS en el código HTML5.

2. Puntos del Proyecto.

  1. Uso de etiquetas HTML5 de forma correcta.
    1. Uso de etiqueta head de forma correcta.
    2. Uso de etiqueta body de forma correcta.
  2. Uso del atributo styles dentro de las distintas etiquetas HTML5 usadas.
  3. El proyecto Web debe de contener como mínimo cinco páginas web HTML, las cuales deben contener:
    1. Página de inicio.
    2. Página de Ventas/Noticias/etc. (Cuatro Productos/Noticias/etc. como mínimo).
      1. Dichos productos/noticias/etc. deberá enlazar con su producto/noticia/etc. detallado.
    3. Página de formulario/registro/etc.
    4. Página de galería de imágenes (diez imágenes como mínimo).
    5. Página de elaboración propia personalizada.
  4. No utilizar etiquetas deprecadas/obsoletas.
  5. Dar formato a las tablas.
  6. SE PUNTÚA MUCHO LA ORIGINALIDAD.

3. Entrega.

La entrega se realizará en la tarea creada para ello.

No se admitirá entregas posteriores a la fecha indicada en dicha tarea creada.

Será entregada la carpeta principal del proyecto comprimida en formato RAR, ZIP, 7ZIP, etc., cualquier entrega realizada que no tenga dicho formato no será evaluado. El nombre de la carpeta debe ser la siguiente estructura Proyecto2Ev-Apellido1Apellido2Nombre.

4. Criterios de Corrección.

Se evaluará que el proyecto contenga todos los puntos descritos en el apartado 2.

Se tendrá en cuenta los siguientes puntos:

  • Elegibilidad y redacción del proyecto (NO VALE COPIAR Y PEGAR DE INTERNET). El copiar y pegar de internet implica tener la calificación de un 0.
  • Buenas prácticas de uso de etiqueta HTML5.
  • Utilización de comentarios para facilitar el entendimiento del código.
  • Prohibido totalmente el uso de Inteligencia Artificial para el Proyecto.
